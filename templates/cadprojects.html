1. Complex Part Modeling

This project focuses on creating highly detailed, single components using advanced feature-based and surface modeling techniques required for precision machining.

Core Objectives:

  • Mastering advanced sketching and constraint application.
  • Utilizing complex features like Lofts, Sweeps, and boundary surfaces.
  • Applying draft, fillets, and shells for manufacturability.
  • Parametric modeling for easy design changes.

Skills Demonstrated:

  • 3D Sketching and multi-plane feature creation.
  • Tolerances and surface finish specification (basic GD&T).
  • Mass property calculation and material assignment.
  • Design validation using feature analysis tools.

2. Hip Implant - SolidWorks Simulation

This project demonstrates SolidWorks modeling and simulation of a hip implant. It includes assembly, constraint definition, and toolpath verification to ensure manufacturability and safe operation.

Core Objectives:

  • Create detailed 3D CAD model of a hip implant in SolidWorks.
  • Simulate assembly with femur component and verify motion constraints.
  • Analyze toolpaths and machining simulation for safe CNC operation.

Skills Demonstrated:

  • SolidWorks 3D part modeling and assembly techniques.
  • Simulation validation including stress and collision checks.
  • CAM toolpath verification and optimization.
  • Applying manufacturing constraints and GD&T basics.

2. Mechanism Assembly & Mates

This project involves combining multiple individual components into a functioning system (e.g., a simple engine, gear train, or clamp) and defining their motion constraints.

Core Objectives:

  • Structuring assembly files and managing sub-assemblies.
  • Using standard mates (coincident, concentric, parallel) correctly.
  • Applying advanced mechanical mates (gear, cam-follower, screw).
  • Defining components to be fixed, rotating, or linear.

Skills Demonstrated:

  • Interference detection and resolving clearance issues.
  • Creating realistic motion simulation paths.
  • Generating Exploded Views for assembly instructions.
  • Utilizing a library of standard parts (e.g., bolts, bearings).

3. Technical Drafting

The objective is to create a complete, industry-compliant 2D blueprint for a manufactured component, ensuring clear communication with production teams.

Video Placeholder: Drafting and Annotation Walkthrough

Core Objectives:

  • Selecting appropriate scale and sheet size.
  • Creating primary, auxiliary, and section views.
  • Properly dimensioning all features (chains, datums).

Skills Demonstrated:

  • Adherence to ANSI/ISO drawing standards.

  • Using Geometric Dimensioning and Tolerancing (GD&T) callouts.

4. CAM Toolpath Generation

This project bridges design and manufacturing by preparing the CAD model for a CNC machine, focusing on efficient tool selection and safe machining strategies.

Video Placeholder: Toolpath Simulation and Optimization

Core Objectives:

  • Defining the machine origin and stock material setup.
  • Selecting correct cutting tools and machining parameters (speeds/feeds).
  • Creating roughing and finishing operations (2D & 3D milling).
  • Simulating the toolpath to check for collisions and material removal.

Skills Demonstrated:

  • Generating and customizing G-code for a specific post-processor.
  • Using rest machining to optimize remaining material removal.
  • Defining complex fixturing and clamping setup.
  • Understanding and applying cutting strategies (e.g., climb vs. conventional).
